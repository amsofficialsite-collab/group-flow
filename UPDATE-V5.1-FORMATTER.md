# GROUP FLOW V5.1 – Facebook Post Formatter

เพิ่มในหน้า Content Library:

- ปุ่ม `จัดโพสต์ให้อ่านง่าย`
- เว้นบรรทัดก่อนหัวข้อและอีโมจิสำคัญ
- แยกรายการย่อยให้อ่านง่ายบนมือถือ
- ตัวอย่างหน้าตาโพสต์แบบ Facebook ก่อนบันทึก
- ข้อความที่จัดแล้วถูกส่งเข้า Daily Queue และ Posting Agent พร้อมบรรทัดเดิม

ไม่ต้องรัน SQL เพิ่ม ใช้ฐานข้อมูลเดิมได้เลย
